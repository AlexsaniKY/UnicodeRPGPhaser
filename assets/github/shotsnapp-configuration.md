# Configuration for shotsnapp images

https://shotsnapp.com

## Big Size

```
width: 1000 px,
height: 800 px,
background fill: #ffffff,
opacity: 0%,
style: chrome,
scale: 75%,
shadow: 0%
```

## Small Size

```
width: 500 px,
height: 400 px,
background fill: #ffffff,
opacity: 0%,
style: chrome,
scale: 36%,
shadow: 0%
```
