# Configuration for shotsnapp images

https://shotsnapp.com

## Big Size

```
width: 1200 px,
height: 800 px,
background fill: #ffffff,
opacity: 0%,
style: chrome,
scale: 75%,
shadow: 20%
```

## Small Size

```
width: 1000 px,
height: 500 px,
background fill: #ffffff,
opacity: 0%,
style: chrome,
scale: 43%,
shadow: 20%
```
